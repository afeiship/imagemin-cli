## installation
```shell
git clone https://github.com/afeiship/imagemin-cli.git
cd imagemin-cli
npm i && npm link
```

## usage
~~~
Usage: imagemin-cli [options]

Options:
  -V, --version  output the version number
  -d, --debug    only show cmds, but not clean.
  -h, --help     display help for command
~~~
